CIS-602 Flask + React Full-Stack Application
🔹 Project Overview
Full-stack web application built using Flask (backend) and React (frontend)
Developed as part of coursework at the University of Massachusetts Dartmouth
Demonstrates integration of REST APIs with a dynamic frontend UI
Implements complete CRUD (Create, Read, Update, Delete) functionality
🔹 Features
Create, view, update, and delete user records
RESTful API endpoints using Flask
Interactive UI built with React
SQLite database for persistent storage
JSON-based communication between frontend and backend
Single-page application behavior
Production build served through Flask
🔹 Tech Stack
Backend: Flask (Python)
Frontend: React (JavaScript)
Database: SQLite
Tools: Git, npm, virtual environment
